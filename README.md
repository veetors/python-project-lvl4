Task Manager is the final project of the Python course of the [Hexlet](https://hexlet.io) education project. It is a small web application that helps you to deal with tasks.

## Start dev-server
```
$ make install

$ cp .env.local .evn

Edit .env

$ make start
```

## Run test
```
$ make test
```
